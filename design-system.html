📋 Principios de Diseño

🎨 Accesibilidad Visual

  • Contraste mínimo de 4.5:1 para texto
  • Tamaño de fuente mínimo de 14px
  • Elementos interactivos de al menos 44x44px

🚀 Animaciones

  • Duración: 200-300ms para micro-interacciones
  • Easing: ease-in-out para suavidad
  • Usar transform y opacity para performance

📱 Responsividad

  • Mobile First: Diseñar primero para móvil
  • Breakpoints: 640px (sm), 768px (md), 1024px (lg), 1280px (xl)
  • Grid flexible con CSS Grid y Flexbox
